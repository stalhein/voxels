# Voxels
Voxels (soon to be renamed) is a work-in-progress voxel engine, possibly becoming a game in the future, made with JS and WebGL2.
## Usage
Go to https://stalhein.github.io/voxels/ to try out.
### Controls
**W** - up,\
**A** - left,\
**S** - down,\
**D** - right,\
**SPACE** - up,\
**SHIFT** - down,\
**LEFT CLICK** - break block,\
**RIGHT CLICK** - place stone.

**Basically Minecraft controls.**
